Прототип дизайна веб-игры Chaotic

По итогам опроса среди участников команды выбрано название "Chaotic!" для реализуемого проекта.

Также был определен логотип, его вы можете увидеть справа вместе с favicon страниц.

На данном этапе стало ясно, что с реализацией логотипа посредством средств CSS могут возни­кнуть проблемы, в силу отсуствия методов отри­совки внешнего контура у текста, кроме хака с множественным выстроением text-shadow.

Принимая во внимание вышесказанное, на данный момент предполагается, в качестве альтернативы обводки текста, использовать заготовленные изо­бражения каждого символа слова "Chaotic!" с альфа-каналом.

Основным шрифтом приложения выбрано семейство Comfortaa

Для анимаций на странице используется библиотека Velocity v2

Некоторые элементы на странице могут использовать Canvas для отрисовки контента

Справа представлена демонстрация ани­мации при входе на сайт неавто­ризированного пользо­вателя. Ани­мация может отличаться от прототипа.
